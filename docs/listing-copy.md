Long description:

```text
Mac Migration Operator Skill is an open Agent Skill for planning, running, validating, and recovering Mac-to-Mac migrations. It guides intake, source freeze, Migration Assistant decisions, first-boot checks, cloud-drive and external-drive review, developer-machine validation, exact-file gap repair, and the final old-Mac retirement gate.

The workflow is built around evidence. It separates confirmed checks from user-confirmed facts, inferences, missing items, and blockers. It keeps the old Mac available as rollback until the new Mac passes validation and the user explicitly approves retirement.

The package includes templates, examples, documentation, and optional read-only macOS helper scripts for users who want stronger local evidence.
```
