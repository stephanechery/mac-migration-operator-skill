# Simple Home Migration

## Scenario

A home user is moving from an older Mac to a new Mac. The important items are personal folders, photos, music, email, browser bookmarks, passwords, and a few everyday apps. The source Mac is working and should stay unchanged until the new Mac is proven ready.

## Recommended Workflow

1. Fill out the intake form with the source Mac, target Mac, transfer method, important folders, core apps, cloud accounts, and any external drives.
2. Freeze the source Mac after checking that recent files and photos are present.
3. Use Migration Assistant, Time Machine, or a trusted backup as the transfer source.
4. On first boot, confirm the user account, Wi-Fi, Apple ID, keychain, browser profile, email, and storage.
5. Record every check in the proof ledger with `confirmed`, `user-confirmed`, `inferred`, `missing`, or `blocked`.
6. Repair gaps by copying only the exact missing file or folder from the source Mac, backup, or cloud service of record.

## Validation Checklist

- Source Mac remains available for rollback and has not been erased, traded in, repurposed, or packed away.
- Desktop, Documents, Downloads, Pictures, Movies, Music, and any custom home folders are checked against the intake form.
- Photos or the user's photo app opens and recent items appear.
- Email opens with the expected account labels, without recording passwords or account secrets.
- Browser bookmarks, profiles, and password manager access are checked without exposing vault contents.
- Critical apps open and recent files appear where expected.
- Any missing item is checked on the source Mac before it is called a migration failure.

## Retirement Gate

Do not retire the source Mac until the proof ledger shows critical folders, photos, email, browser data, password manager access, and core apps as `confirmed` or intentionally deferred. The retirement packet should list any `missing` or `blocked` items and the user should approve the remaining risk in plain language before any source Mac erase, trade-in, donation, storage, or repurpose step.
