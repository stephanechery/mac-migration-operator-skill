# External Drives

External drives need their own validation. A successful Mac transfer does not prove that external-drive folders, media libraries, or backups are complete.

## Inventory Drives

1. List each external drive by visible name.
2. Record what it is used for.
3. Confirm whether it is a live work drive, backup drive, archive drive, or transfer drive.
4. Record whether the old Mac, new Mac, or both Macs can mount it.

## Check Format and Permissions

1. Mount the drive on the new Mac.
2. Open the top-level folders.
3. Open one representative file from each important folder.
4. Confirm the user can read expected files.
5. If write access matters, ask before creating any test file.

## Compare Expected Folders

Use the intake form to list expected folder names. Compare those names against the mounted drive. If a folder is missing on the new Mac, check the old Mac before deciding whether the drive or transfer failed.

## Media Libraries

Photo, video, and music libraries can be large and app-specific. Open the app that owns the library and confirm recent items appear. Do not move a library package unless the user has approved that exact step.

## Backups

If the external drive is a backup, do not treat it as a working copy. Confirm that the backup exists and can be browsed, but keep repair steps separate from backup validation.
