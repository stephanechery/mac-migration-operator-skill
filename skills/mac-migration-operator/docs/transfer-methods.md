# Transfer Methods

Choose the transfer method based on what the user needs to preserve, what equipment is available, and how much risk the user can tolerate.

## Migration Assistant

Use Migration Assistant when both Macs are available and the goal is a broad transfer of users, apps, settings, and files.

Best for:

- Home users moving to a new Mac.
- Users who want the same account and settings on the new Mac.
- Moves where the old Mac still works.

Watch for:

- Apps that need fresh sign-in.
- Old Intel apps on Apple silicon.
- Large transfers over slow wireless networks.

## Time Machine

Use a Time Machine backup when the source Mac is not convenient to use directly or when a recent backup is more reliable than the live machine.

Best for:

- Source Macs that are unstable.
- Users with a complete recent backup.
- Recovery after hardware problems.

Watch for:

- Backups that are old or incomplete.
- External drives that were excluded from backup.
- Restores that bring old settings the user no longer wants.

## Cloud Drive

Use cloud sync for files that already live in iCloud Drive, Google Drive, Dropbox, OneDrive, or another sync tool.

Best for:

- Documents that are already cloud-managed.
- Users who need a clean Mac setup.
- Small transfers with good internet.

Watch for:

- Online-only placeholders.
- Files that are still uploading on the old Mac.
- Multiple accounts signed into the same app.

## External Drive

Use an external drive when files are too large for cloud sync or when the user needs a controlled manual copy.

Best for:

- Photos, videos, archives, and project folders.
- Slow or limited internet.
- A staged transfer with checksums.

Watch for:

- Copying from an old or failing drive.
- Duplicate folders with similar names.
- Permission problems after copy.

## Manual Copy

Use manual copy for small, known sets of files. Prefer exact paths and a proof ledger.

Best for:

- A few important folders.
- Repairing gaps after a larger transfer.
- Moving files from a known backup.

Watch for:

- Broad folder overwrites.
- Missing hidden app data.
- Confusion between old and new folder copies.

## Hybrid Plan

Many migrations use more than one method. A safe hybrid plan might use Migration Assistant for the user account, cloud sync for cloud documents, and an external drive for large media.
